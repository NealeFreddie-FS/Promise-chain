About Pokemon Explorer

Pokemon Explorer is a web application that allows you to discover and learn about the original 151 Pokemon from the Kanto region. The application uses the Pokemon API to fetch data about each Pokemon, including their types, habitats, physical characteristics, and descriptions. You can save your favorite Pokemon to your personal collection for easy reference.

This application demonstrates the use of asynchronous JavaScript with fetch and async/await to interact with the Pokemon API. It makes multiple API calls to retrieve comprehensive data about each Pokemon, combining information from different endpoints.

Features:

  • Explore random Pokemon from the original 151
  • View detailed information about each Pokemon
  • Save your favorite Pokemon to your collection
  • Responsive design for all devices
  • Beautiful and intuitive user interface
